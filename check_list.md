<새로운 랜딩 제작시 체크리스트>
1. 지점 픽셀 삽입 확인
2. GTM 코드 삽입 확인(GTM head/body)와 메타픽셀만 안티에 알려주면 알아서 세팅함!
3. 메타 이벤트 생성
4. 지점위치 확인/ 센터내부 둘러보기 링크연결 확인
5. GTM 변수, 트리서, 테그 설정하기

<광고등록시 확인사항>
1. 지점 위치 반경 및 맞춤타겟/ 제외타겟 재확인
2. 광고소재 등록후 3일차 4일차 확인후 확장및 off 결정하기
3. "게제되지 않은 사항" 꼭 확인후 처리할것
4. 광고소재 등록시 한 세트에 묶어도 되는지 정확히 판단할것

< 광고 소재 제작시 주의할점>
1. 인지단계 : 후킹이 강한 배너나 영상을 이용해 랜딩으로 유입
2. 고려단계 : 신뢰와 공감주는 소재 -(비포/에프터 사진, 실제회원 운동영상, 트레이너 전문적 자세교정 수업영상, 깨끗한 샤워실등 헬스장 내부사진)
3. 검색단계: 행사 마감임박, 지금결제시 증정품 소개... 상실 고포를 자극하는 소재
4. 전환 : 재등록 혜택안내, 바디챌린지/ 바디프로필 반 모집/ 지인추천 이벤트 소재

<랜딩제작시 주의할점>
0. 랜딩에 히어로 섹션은 광고에서 본 그 메인 텍스트로 시작해야 이탈율이 적다
1. 광고에서 감정에 호소했다면 랜딩에서는 논리로 설득하자!
(실제 운동사진/ 트레이너 PT사진/ 후기사진...)
2. 고객의 궁금증 모두 풀어줘라.(FAQ로 먼저 풀어라)
3. 지금 결제 해야만 하는 혜택보다는 기회상실을 더 강조

<유지보수 및 관리>
1. 특정 페이지 서버 캐시 삭제 (인스타 브라우저 등에서 변경사항 적용 안 될 때)
- 인터넷 주소창에 아래와 같이 입력하고 접속
- 형식: `https://mindfitness-[지점도메인].com/api/revalidate?path=/[랜딩경로]&secret=mindfitness123!`
